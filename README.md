# 简介

单体式架构

- Spring Boot 3.x
- Filter（安全验证过滤器）
- Redis

# 项目结构

```text
├ monolithic
├── biz        （业务模块）
├── scheduling （任务调度模块）
├── web        （Web模块）
└── websocket  （WebSocket模块）
```
